# 搭把手 - 需求清单与功能说明

## 项目简介

**搭把手**是一个校园共享技能互助平台，让同学们可以把自己的闲置技能和碎片时间变成互助资源。在这里，你可以用自己的特长帮助别人，也可以找到需要的帮助，用"时间币"代替金钱交易，让互助变得更轻松、更有温度。

---

## 核心理念

- **技能换时间**：用你的特长换取别人的帮助时间
- **时间换服务**：用积累的时间币获得需要的服务
- **无压力互助**：虚拟积分消除金钱交易的心理压力
- **信任机制**：完善的信用体系保障每次互助

## 核心业务流程

### 1. 技能服务流程
```
发布技能 → 设置闲时 → 用户下单 → 积分担保 → 线下履约 → 扫码核销 → 积分结算 → 双向评价
```

### 2. 求助需求流程
```
发布需求 → 设置悬赏 → 智能匹配 → 服务方接单 → 积分担保 → 线下履约 → 扫码核销 → 积分结算 → 双向评价
```

### 3. 订单状态流转
```
待支付 → 已支付(担保中) → 服务中 → 待确认 → 已完成
   ↓         ↓           ↓        ↓
 已取消    已退款      已退款    争议中
```

## 特色功能

### 1. 闲时格子日历
可视化展示个人空闲时间，方便他人预约，避免时间冲突。

### 2. 智能匹配推荐
基于地理位置、技能标签、信任分等多维度智能推荐匹配。

### 3. 动态核销码
线下服务完成后生成动态核销码，扫码确认，防伪防截图。

### 4. 超时熔断机制
完善的超时处理机制，保障双方权益，避免"放鸽子"。

### 5. 信任分体系
根据历史表现自动计算信任分，高信任分享优先推荐。

### 6. 技能置换
支持"以技能换技能"的非现金交易模式，降低心理压力。

## 使用场景示例

### 场景1：PPT排版帮助
- 小明擅长PPT设计，在平台发布"PPT排版"技能
- 设置本周空闲时段：周二下午、周四晚上
- 小红需要做课程PPT，在看板看到小明的服务
- 小红下单预约周二下午，积分自动冻结
- 周二下午小明帮小红做完PPT
- 小明展示核销码，小红扫码确认
- 积分自动结算，双方互评

### 场景2：急求正装领带
- 小华明天有面试，急需正装领带
- 在平台发布"急求正装领带"需求，设置12小时悬赏
- 系统匹配到同校区有领带的小李
- 小李接单，积分冻结
- 晚上小李把领带借给小华
- 小华扫码确认，积分结算
- 双方互评，记录信用

### 场景3：技能置换
- 小张会弹吉他，想学摄影
- 小王会摄影，想学吉他
- 系统匹配到双方的置换需求
- 双方协商：小张教小王2小时吉他，小王教小张2小时摄影
- 完成后互相确认，记录信用

## 常见问题

### Q1：积分从哪里来？
A：注册时赠送初始积分，完成任务、获得好评、签到活跃都可以获得积分。

### Q2：积分会过期吗？
A：积分永久有效，不会过期。

### Q3：如何提高信任分？
A：按时完成订单、获得好评、积极参与都可以提高信任分。

### Q4：被放鸽子怎么办？
A：系统有超时熔断机制，超时自动退款并记录对方违规。

### Q5：可以取消订单吗？
A：可以取消，但根据取消时间可能扣除一定积分。

### Q6：如何联系对方？
A：通过平台内置的即时聊天功能联系对方。

### Q7：评价可以修改吗？
A：评价一旦提交不可修改，保障评价的真实性。

### Q8：遇到争议怎么办？
A：可以发起申诉，由客服审核处理。
