# Tab Bar Icons

需要 8 个图标文件（普通态 + 选中态），建议尺寸 81x81 px：

| 图标 | 普通态 | 选中态 |
|------|--------|--------|
| 首页 | home.png | home-active.png |
| 发布 | publish.png | publish-active.png |
| 消息 | message.png | message-active.png |
| 我的 | mine.png | mine-active.png |

推荐资源：
- TDesign 图标库: https://tdesign.tencent.com/design/icon
- 微信图标规范: 建议使用纯色填充，选中态使用主题色 #0052d9

在微信开发者工具中导入图标后，放到此目录即可。
